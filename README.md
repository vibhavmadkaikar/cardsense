# CardSense

**Smarter decisions for every card you use.**

CardSense is a personal finance platform that helps users manage cards
(credit and debit), track spending, understand benefits, and decide
which card to use for maximum value.

## Problem Statement

Managing multiple cards is hard:

- Different reward structures
- Missed benefits
- Late payments
- No clear guidance on which card to use

CardSense solves this with a data-driven approach.

## Key Features (Planned)

- Card management (credit & debit)
- Transaction tracking
- Benefits & rewards explorer
- Smart card recommendation engine
- Payment reminders
- Spending analytics

## Tech Stack

- **Frontend:** Flutter (Android, iOS, Web)
- **Backend:** NestJS (TypeScript)
- **Database:** PostgreSQL, Redis
- **Cloud:** AWS (ECS, RDS, S3, Lambda)

## Architecture

- Modular Monolith (MVP)
- Microservice-ready design
- Security-first approach
